# Bus Sim 2018 🚍


### Get started
install all the dependencies

`npm i`

to run the tests type

 `npm test`
 
 --
 
 ### Start the application
 
##### docker

 `make up`
 
##### node
 
 `NODE_ENV=production node app.js` 
 
  or development mode
  `node app.js`
 
 ##### npm
 
 dev `npm dev`
 production `npm prod`
